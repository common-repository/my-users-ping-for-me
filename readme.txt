== Installation ==

1. Upload `my-users-ping-for-me` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the "Plugins" menu in WordPress
1. Visit your my-users-ping-for-me (*Settings my-users-ping-for-me Settings*)
1. Configure options
1. That"s it!
